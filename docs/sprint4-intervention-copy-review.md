## Review checklist (per message)

| # | Criterion | Pass test |
|---|-----------|-----------|
| 1 | **Warm, not condescending** | reads as a supportive nudge, not a scold ("Let's take another look" ✓ / "You weren't paying attention" ✗) |
| 2 | **Concise** | **< 15 words** |
| 3 | **Action-oriented** | tells the student what to *do* next (re-read, refocus, rest, ask) |
| 4 | **Type-appropriate** | distraction → refocus; confusion → clarify/revisit; fatigue → rest/break |
| 5 | **No clinical/▾DNA language** | no IQ/EQ/diagnostic claims (PRD §18 / CLAUDE.md) |

A message **passes** only if all 5 hold. Any fail → fix request to Dev 1.

                review(pkg_id, i, itype, j, m)   # apply the 5-point checklist
